Installation:
-------------

* Install Hammerspoon
* Install Karabiner-Elements
* Put `hammerspoon/init.lua` in `~/.hammerspoon/init.lua`
* Put `karabiner/karabiner.json` in `~/.config/karabiner/karabiner.json`

That’s pretty much it.
